Program SADAR #4: Mobil Pajak Keliling sebagai Pusat Layanan Terpadu. Transformasi mobil pajak keliling dari satu fungsi menjadi unit multi-layanan: pembayaran pajak digabungkan dengan layanan kesehatan/kependudukan agar masyarakat memiliki alasan tambahan untuk datang. Dilengkapi terminal pembayaran luring-ke-daring yang menyimpan data transaksi sementara saat sinyal tidak tersedia dan sinkronisasi saat sinyal kembali. Segmen sasaran: S1, O1 di hinterland. Pemangku: SAMSAT, Kelurahan. Tipologi: Wilayah Hinterland.
