Program SADAR #9: Audit Kepatuhan Kendaraan Dinas. Penerapan kebijakan 'Tidak Bayar Pajak, Tidak Ada BBM': kendaraan dinas yang menunggak pajak tidak mendapat kuota bahan bakar bersubsidi atau anggaran pemeliharaan dari APBD. Bertujuan memberikan keteladanan kepada masyarakat umum. Segmen sasaran: M2 sub-segment kendaraan dinas. Pemangku: Bapenda, Pemerintah Daerah. Tipologi: semua.
