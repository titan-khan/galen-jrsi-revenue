# framework_v1.4_anti_pattern — anti_confuse_s1_s2

**Source:** `framework_v1.4_anti_pattern`  
**Category:** `anti_confuse_s1_s2`

---

ANTI-PATTERN KRITIS: JANGAN confuse S1 dan S2. Keduanya sama-sama 'tidak pernah transact' (tanggal_transaksi NULL), tapi treatment beda 180 derajat. S1 = kendaraan tangan kedua tanpa balik nama, usia ≤15 tahun, hampir semua sepeda motor (97.5%). PENGHALANG = biaya BBNKB. Treatment = pengurangan BBNKB + community campaign + Keliling dengan kapabilitas balik nama. Setiap konversi = aliran pendapatan baru BERULANG. S2 = kemungkinan tidak ada fisik, usia >15 tahun atau >5 tahun tunggakan + >20 tahun usia. Treatment = data cleanup, BUKAN kampanye. Salah klasifikasi → kirim amnesti ke S2 (sia-sia, merusak kredibilitas) atau kirim cleanup notice ke S1 (kehilangan opportunity revenue baru).
