Program SADAR #7: Kemitraan Korporat SAMSAT. Layanan pemeriksaan di lokasi perusahaan untuk armada perkebunan kelapa sawit dan pertambangan. Pemeriksaan fisik massal dan pembayaran kolektif langsung di area operasional. Dorongan melalui pemberian label 'Perusahaan Taat Pajak' sebagai komponen penilaian tata kelola lingkungan dan sosial perusahaan (ESG). Segmen sasaran: M1, M2 (untuk fleet). Pemangku: SAMSAT, Bapenda. Tipologi: Hub Industri.
