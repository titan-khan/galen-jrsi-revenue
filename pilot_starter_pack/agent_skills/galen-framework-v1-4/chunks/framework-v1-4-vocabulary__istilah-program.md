Vocabulary program SADAR (Bahasa daerah Dayak): 'Hayak Bahayau' = nama program Duta Pajak kelurahan (artinya 'kebersamaan' dalam bahasa Dayak Ngaju). 'Huma Betang Mahaga' = nama alternatif Duta Pajak (Huma Betang = rumah panjang Dayak, Mahaga = melestarikan). 'Taka' = mari/ayo (bahasa Dayak), digunakan dalam tagline pengingat: 'Taka bayar pajak kendaraan, agari Kalteng makin maju'. 'agari' = supaya (bahasa Dayak). Penggunaan bahasa daerah dalam pesan WhatsApp ke H1 dan K1 untuk membangun kedekatan emosional.
