Program SADAR #5: Amnesti Pajak Bertarget. Program amnesti yang DITARGETKAN secara spesifik per segmen berdasarkan analisis data: pengurangan denda parsial 50-75% untuk M1, penghapusan denda penuh untuk M2, pengurangan bea balik nama (BBNKB) untuk S1. SELALU dibatasi waktu. TIDAK PERNAH bersifat menyeluruh tanpa pandang bulu. WAJIB diikuti penegakan setelah batas waktu berakhir. Segmen sasaran: M1 (parsial), M2 (penuh denda), S1 (BBNKB). Pemangku: Bapenda, SAMSAT. Tipologi: semua wilayah.
