Decision rule untuk channel routing berdasarkan phone availability. >95% punya HP = digital-first OK (H1, K1, O1, M1 di Pusat Urban). 50-95% = hybrid mandatory (M2: 71% — gabung WhatsApp + kelurahan/RT-RW + surat fisik). <25% = community-based only (S1: 2%, S2: 19%). Untuk M2, 29% offline tidak bisa di-skip — perlu koordinasi kelurahan + RT/RW + surat fisik untuk reach. Untuk S1, BUKAN digital — community-based via Duta Pajak, RT/RW, SAMSAT Keliling. Untuk S2, NO campaign — data cleanup only.
