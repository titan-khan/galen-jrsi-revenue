Program SADAR #2: Program Loyalitas dan Insentif. Penghargaan bagi WP patuh konsisten. Sertifikat digital 'Wajib Pajak Taat', pengurangan pajak tahun berikutnya untuk pembayaran lebih dari 30 hari sebelum jatuh tempo, manfaat keselamatan dari Jasa Raharja berupa potongan harga di mitra terkait, prioritas layanan. Bertujuan membangun hubungan positif antara WP dan otoritas pajak. Segmen sasaran: H1. Pemangku: Jasa Raharja, Bapenda. Tipologi: semua wilayah.
