Program SADAR #6: Kolaborasi Ekosistem. Aktivasi peran setiap lini: Polri via kamera ETLE terintegrasi dengan data pajak, Bapenda via Duta Pajak di tingkat kelurahan (Hayak Bahayau atau Huma Betang Mahaga), Jasa Raharja via manfaat keselamatan bagi yang taat, mitra dompet elektronik dan perbankan untuk skema cicilan pajak. Segmen sasaran: O1, M1, M2, S1, S2. Pemangku: semua. Tipologi: semua.
