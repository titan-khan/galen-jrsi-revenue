Decision rule untuk RACI amnesti. Regulasi amnesti = Bapenda Accountable, Jasa Raharja Consulted (untuk komponen SWDKLLJ), SAMSAT Consulted (untuk eksekusi loket), Polri Consulted (untuk koordinasi razia pasca-amnesti). Eksekusi WhatsApp campaign = Vendor TI/Gateway WA Responsible, Bapenda Informed. Eksekusi razia pasca-amnesti = Polri Responsible, SAMSAT Consulted (untuk pembayaran on-site). Eksekusi 'Tidak Bayar = Tidak Ada BBM' kendaraan dinas = Bapenda Accountable, Pemerintah Daerah Consulted. Aktivasi Duta Pajak = Kelurahan Accountable, SAMSAT Responsible (rekrutmen + training).
