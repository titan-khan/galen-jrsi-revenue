# framework_v1.4_anti_pattern — anti_lupa_razia_pasca_amnesti

**Source:** `framework_v1.4_anti_pattern`  
**Category:** `anti_lupa_razia_pasca_amnesti`

---

ANTI-PATTERN KRITIS: Amnesti TANPA razia pasca-deadline. Kalau amnesti penuh denda M2 berakhir tanpa diikuti razia, pesan ke seluruh ekosistem adalah 'tidak ada konsekuensi'. Pemilik kendaraan yang tidak memanfaatkan amnesti akan tetap tidak bayar — tahu bahwa siklus berikutnya akan ada amnesti lagi. Ini menghancurkan deterrent dan menciptakan dependency culture. Solusi: setiap regulasi amnesti M2 wajib include rencana operasional razia + tilang setelah deadline berakhir, dengan jadwal dan lokasi targeted berbasis data konsentrasi M2 yang tidak respond. Polri/Ditlantas adalah Responsible. Bapenda Accountable untuk regulasi pasca-amnesti.
