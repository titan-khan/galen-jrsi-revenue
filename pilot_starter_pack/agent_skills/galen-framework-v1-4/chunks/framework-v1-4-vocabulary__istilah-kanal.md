# framework_v1.4_vocabulary — istilah_kanal

**Source:** `framework_v1.4_vocabulary`  
**Category:** `istilah_kanal`

---

Vocabulary kanal layanan: 'SAMSAT' = Sistem Administrasi Manunggal Satu Atap (one-stop kantor pajak kendaraan). 'SAMSAT Keliling' = mobile SAMSAT yang berkeliling kecamatan/desa. 'SAMKEL' = nama lain SAMSAT Keliling. 'BUMDES' = Badan Usaha Milik Desa (untuk kolaborasi titik layanan rural). 'BRILink' = agen perbankan BRI (untuk pembayaran rural). 'PT POS' = Pos Indonesia (titik layanan rural). 'ETLE' = Electronic Traffic Law Enforcement (kamera pelanggaran lalu lintas terintegrasi). 'WA Gateway' = WhatsApp Business API gateway untuk kirim massal. 'Duta Pajak' = volunteer kelurahan untuk sosialisasi pajak.
