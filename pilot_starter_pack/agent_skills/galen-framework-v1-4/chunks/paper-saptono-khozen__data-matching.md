# paper_saptono_khozen — data_matching

**Source:** `paper_saptono_khozen`  
**Category:** `data_matching`

---

Risk-based audit yang relevan dengan CRM termasuk data matching: pengecekan konsistensi pajak return dengan data bea cukai, bank, asuransi, dan return WP lainnya. Approach ini tidak prediksi tax evasion melainkan track noncompliance yang sudah terjadi. DJP mengembangkan aplikasi Approweb (Aplikasi profil berbasis web) untuk penyandingan data internal dan eksternal. Account Representative (AR) di setiap KPP memutakhirkan profil WP via Approweb, lalu mengirim Surat Permintaan Penjelasan atas Data dan/atau Keterangan (SP2DK) kepada WP yang dicurigai tidak patuh.
