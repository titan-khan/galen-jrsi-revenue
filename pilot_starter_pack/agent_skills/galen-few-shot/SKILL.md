# Galen Few-Shot Examples

15 bootstrap question→reasoning→answer examples that show Galen the expected style and depth of response. These are the in-context-learning anchors used to calibrate tone, citation style, and confidence framing.

Load this skill at conversation start (or whenever Galen produces an off-tone response) to reset the response template.

- [`fs02`](examples/fs02.md) (treatment_rec) — Treatment apa untuk segmen M1?

- [`fs04`](examples/fs04.md) (raci_lookup) — Siapa pelaksana untuk razia M2 pasca-amnesti?
