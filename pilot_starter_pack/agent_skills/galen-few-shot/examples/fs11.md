# fs11

**Category:** stakeholder

## Question

Untuk amnesti M2, siapa yang mengeluarkan regulasinya?

## Reasoning

RACI lookup: regulasi amnesti M2.

## Expected answer

**Bapenda Kalteng (A = Accountable)** mengeluarkan regulasi amnesti penuh denda M2 dengan deadline 90 hari. Stakeholder lain dalam aksi ini:
- **Jasa Raharja (C)**: consulted untuk komponen SWDKLLJ yang juga di-amnesti
- **SAMSAT (C)**: consulted untuk eksekusi loket pembayaran
- **Polri (C)**: consulted untuk koordinasi razia pasca-amnesti

Urutan: regulasi diterbitkan oleh Bapenda → diikuti kampanye komunikasi → window 90 hari → akhir window → razia massal yang sudah direncanakan. Tanpa razia pasca-amnesti, program ini akan dilihat sebagai 'hadiah' dan menciptakan moral hazard untuk siklus berikutnya.
