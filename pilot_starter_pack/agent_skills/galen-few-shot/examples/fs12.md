# fs12

**Category:** regional_strategy

## Question

Bagaimana strategi untuk hub industri seperti Sampit?

## Reasoning

Hub industri belum ada data, tapi kerangka strategi tersedia.

## Expected answer

**Catatan limitasi: data kuantitatif untuk Hub Industri (Kotim/Sampit, Kobar, Bartim, Barut) belum tersedia di pilot saat ini.** Kerangka strategi yang tersedia:

Karakteristik Hub Industri:
- Didominasi armada perusahaan perkebunan kelapa sawit dan pertambangan
- Banyak kendaraan dengan plat luar daerah (non-KH)
- Persoalan mutasi plat nomor yang kompleks

Strategi differentiated dari Pusat Urban:
1. **Kemitraan Korporat SAMSAT** — pemeriksaan di lokasi perusahaan, bukan menarik kendaraan ke kantor SAMSAT
2. **Pemeriksaan fisik massal** + pembayaran kolektif langsung di area operasional perkebunan/pertambangan
3. **Tekanan via skor ESG** — label 'Perusahaan Taat Pajak' sebagai komponen penilaian tata kelola
4. **WhatsApp untuk armada** — koordinasi dengan bagian keuangan perusahaan, BUKAN individual driver
5. **Negosiasi pembayaran kolektif** untuk fleet dengan tunggakan

Untuk pilot ini, recommendation: extend data pipeline ke kabupaten Hub Industri sebagai prioritas berikutnya setelah PR pilot lulus.
