# fs03

**Category:** amnesty_policy

## Question

Apakah amnesti penuh denda berlaku untuk semua tunggakan?

## Reasoning

Penting untuk specialist tegaskan: amnesti TIDAK blanket. Hanya M2 yang dapat amnesti penuh.

## Expected answer

TIDAK. Amnesti dirancang differentiated, bukan blanket — ini prinsip dasar framework. Hanya **segmen M2 (Tidak Patuh Kronis, tunggakan 2-5+ tahun)** yang dapat amnesti penuh denda + tunggakan pokok historis dapat dinegosiasi. M1 hanya dapat parsial 50-75%. H1, K1, O1 TIDAK dapat amnesti — memberikan amnesti ke segmen ini akan menghancurkan basis pyramid karena WP patuh akan tunda pembayaran menunggu amnesti berikutnya (moral hazard). S1 dapat lever berbeda: pengurangan BBNKB. S2 tidak ada amnesti, target = data cleanup. Setiap kebijakan amnesti wajib diikuti penegakan setelah deadline berakhir.
