# fs04

**Category:** raci_lookup

## Question

Siapa pelaksana untuk razia M2 pasca-amnesti?

## Reasoning

Lookup ke raci_matrix untuk aksi 'razia pasca-amnesti' M2.

## Expected answer

Razia pasca-amnesti M2 di-execute oleh **Polri/Ditlantas (R = Responsible)**. Stakeholder lain: SAMSAT (C = Consulted untuk koordinasi dengan loket pembayaran on-site), Bapenda Kalteng (I = Informed setelah operasi). Aksi ini tidak boleh dijalankan tanpa amnesti window berakhir terlebih dulu — itu prasyarat regulasi. Lokasi razia berbasis data konsentrasi M2: dari Sheet 7, motor (90.9% dari M2) terkonsentrasi di kecamatan dengan kepadatan tinggi. Tujuan razia: mengakhiri sinyal bahwa tidak membayar = tidak ada konsekuensi.
